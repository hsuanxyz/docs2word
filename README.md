# docs2word(自用型)

📖 将文档导按词频提取出单词，然后背背背 😂 

数据来源(2017-5) 主流前端库文档、MDN 文档、Stackoverflow 3000 问、Github TOP 100 > Issues TOP 100。

## 怎么用？

使用纸笔：在线的单词表 [words.hsuan.xyz](https://words.hsuan.xyz)
